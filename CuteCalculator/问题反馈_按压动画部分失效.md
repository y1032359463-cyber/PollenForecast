# Grid+ForEach 中部分按钮 scale 动画不生效

## 环境
- DevEco Studio 6.0.1 Beta1
- HarmonyOS SDK 6.0.0.47 (API 20)
- 真机测试

## 问题
19个按钮使用相同组件代码，**7个按钮的 `@State` 变化不触发 scale 动画**。

### 异常按钮（7个）
```
第1行: [0]C  [1]del  [2]%  [3]÷  ← 全部无动画
第3行: [9]5                      ← 无动画
第5行: [17].  [18]=              ← 无动画
```

### 正常按钮（12个）
7, 8, 9, 4, 6, 1, 2, 3, 0, ×, -, +

---

## 代码

### 按钮组件
```typescript
@Component
export struct ImageButton {
  @Prop label: string = '';
  @State private scaleValue: number = 1;

  build() {
    Stack() {
      Image($rawfile(`btn_${this.label}_normal.png`))
        .width('100%').height('100%')
        .objectFit(ImageFit.Contain)
    }
    .width('100%').height('100%')
    .scale({ x: this.scaleValue, y: this.scaleValue })
    .onTouch((event: TouchEvent) => {
      if (event.type === TouchType.Down) {
        animateTo({ duration: 80 }, () => { this.scaleValue = 0.92; });
      } else if (event.type === TouchType.Up || event.type === TouchType.Cancel) {
        animateTo({ duration: 120 }, () => { this.scaleValue = 1; });
      }
    })
  }
}
```

### 键盘组件
```typescript
@Component
export struct Keyboard {
  private buttons: string[] = ['C','del','%','÷','7','8','9','×','4','5','6','-','1','2','3','+','0','.','='];

  build() {
    Grid() {
      ForEach(this.buttons, (item: string, index: number) => {
        GridItem() {
          ImageButton({ label: item })
        }
      }, (item: string, index: number) => `${item}_${index}`)
    }
    .columnsTemplate('1fr 1fr 1fr 1fr')
    .rowsTemplate('1fr 1fr 1fr 1fr 1fr')
  }
}
```

## 现象
- ✅ 触摸事件正常触发
- ✅ `scaleValue` 值确实改变（日志确认）
- ❌ 部分按钮 UI 不重绘

为什么相同组件在 Grid+ForEach 中表现不一致？是否与组件复用机制有关？

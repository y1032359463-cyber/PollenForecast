## 📱 应用简介(备案用)

**一句话描述(30字内)**:
```
提供花粉浓度预报的工具类应用,帮助过敏人群提前防护
```

**详细描述(100-300字)**:
```
花粉浓度播报是一款专为花粉过敏人群打造的健康工具应用,提供精准的花粉浓度预报服务。

核心功能:
1. 实时花粉浓度查询 - 支持全国23+重点城市
2. 5日预报趋势 - 折线图展示未来花粉变化
3. GPS定位服务 - 自动获取当前位置信息
4. 地图可视化 - 基于华为Map Kit的地图展示
5. 个性化设置 - 震动反馈、深色模式等

应用特点:
- 无用户注册
- 无社交功能
- 无用户生成内容(UGC)
- 仅查询展示第三方API数据
- 数据本地缓存,离线可查看历史

数据来源: Google Pollen API
地图服务: 华为Map Kit
```

# 如何查看新UI效果 - 解决方案指南

## ❌ 遇到的问题

```powershell
.\hvigorw : 无法将".\hvigorw"项识别为 cmdlet、函数、脚本文件或可运行程序的名称
```

**原因**: 项目缺少`hvigorw`构建脚本,需要通过DevEco Studio IDE来构建运行。

---

## ✅ 解决方案

### 方法1: 使用DevEco Studio (最推荐) ⭐

**步骤**:
1. 打开DevEco Studio
2. File → Open → 选择 `C:\HarmonyOS_App_Plans\Healthy_life`
3. 等待索引完成
4. 连接设备或启动模拟器
5. 点击运行按钮 ▶️ 或按 `Shift + F10`

**您就能看到**:
- 📱 2列Grid布局的任务卡片
- 🎨 彩色emoji图标 (48px圆形)
- 📊 渐变进度条 (6px高)
- ✨ 点击缩放动画

---

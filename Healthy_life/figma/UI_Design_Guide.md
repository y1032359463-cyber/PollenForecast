## 🎭 组件设计

### 1. 任务卡片 (TaskCard)

```typescript
// 现代化设计
- 白色背景 + 轻微阴影
- 左侧: 彩色图标(渐变圆形背景)
- 中间: 任务名 + 进度条
- 右侧: 完成数量 + 单位
- 底部: 微小的时间文字
```

### 2. 进度条 (ProgressBar)

```typescript
// 渐变进度条
- 高度: 6px
- 圆角: 3px
- 背景: #F0F0F0
- 进度: 渐变色(primaryLight → primary)
- 动画: 0.3s ease
```

### 3. 按钮 (Button)

```typescript
// 主按钮
- 背景: 渐变(primary → primaryLight)
- 圆角: 24px (胶囊形)
- 高度: 48px
- 阴影: 轻微
- 按下: 轻微缩小动画

// 次按钮
- 背景: 透明
- 边框: 1px solid primary
- 文字: primary
```

## 🎯 设计原则

1. **一致性**: 所有组件遵循统一的设计语言
2. **可读性**: 足够的对比度和字号
3. **响应式**: 适配不同屏幕尺寸
4. **可访问性**: 颜色盲友好,触摸目标足够大
5. **性能**: 避免过度动画,优化渲染

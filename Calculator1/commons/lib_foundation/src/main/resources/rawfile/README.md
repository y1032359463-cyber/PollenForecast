# 音频资源文件说明

## 目录说明
此目录用于存放应用的原始音频资源文件（rawfile）。

## 所需文件
- `click.mp3` - 按键点击音效文件

## 音频文件要求
- **格式**: MP3 / OGG / WAV
- **时长**: 建议 50-200ms (短促音效)
- **音质**: 建议 44.1kHz 采样率, 单声道或立体声
- **大小**: 建议不超过 50KB

## 获取音效文件的方法
1. **系统音效库**: 从 Android/iOS 系统音效中提取
2. **开源音效**: 从 [freesound.org](https://freesound.org) 等网站下载
3. **自制音效**: 使用 Audacity 等工具录制或生成
4. **推荐音效**: 搜索 "UI click sound" 或 "button tap sound"

## 降级方案
如果音频文件不存在，代码会自动降级为使用**短促振动(15ms)**来模拟按键音效，不会影响应用运行。

## 使用示例音效
可以从以下位置找到合适的点击音效:
- Windows: `C:\Windows\Media\Windows Notify.wav`
- 或使用任何简短的"滴答"、"咔哒"声音

将音频文件重命名为 `click.mp3` 并放置在此目录即可。

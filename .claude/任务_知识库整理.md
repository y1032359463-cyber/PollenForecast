# 📋 知识库整理任务

> **使用方法**: 在对话中输入 `@任务_知识库整理.md` 即可执行

---

## 执行步骤

请按顺序执行以下3个步骤：

### 1️⃣ 迁移已验证知识
将 `@知识库_中转站（待验证实践效果）.md` 中**已经实践出结果**的知识整理进 `@知识库.md`
- 成功验证的 → 添加到知识库对应分类
- 验证失败的 → 添加到避坑指南

### 2️⃣ 整理专家回答
将 `@当前问题.md` 中的专家回答/知识点整理进 `@知识库_中转站（待验证实践效果）.md`
- 保留代码示例
- 标注专家来源
- 标记待验证状态

### 3️⃣ 制定下一步计划
根据 `@当前问题.md` 中专家的回答：
- 汇总分析各专家观点
- 找出共识和分歧点
- 制定具体修复/实施计划
- 更新待验证清单优先级

---

## 📁 涉及文件

| 文件 | 作用 |
|------|------|
| `.claude/知识库.md` | 主索引，已验证知识 |
| `.claude/知识库_中转站（待验证实践效果）.md` | 待验证方案暂存 |
| `.claude/当前问题.md` | 当前问题+专家回复 |
| `.claude/知识库/` | 分类知识库文件夹 |

---

## ⚡ 快捷用法

**完整执行**（推荐）：
```
执行 @任务_知识库整理.md
```

**单独执行某一步**：
```
执行 @任务_知识库整理.md 的第1步
```

**带专家回答数量**：
```
@当前问题.md 有4个专家回答，执行 @任务_知识库整理.md
```

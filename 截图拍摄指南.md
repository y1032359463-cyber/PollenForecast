## 🎨 截图美化建议

### 设备选择
- **推荐**: 使用真机截图（更真实）
- **备选**: 使用 DevEco Studio Previewer（像素完美）

### 时间和状态栏
- 时间显示: 建议 10:30 或 14:30（避免 00:00、12:00）
- 电量显示: 70-80%（避免低电量红色图标）
- 网络信号: 满格或 4 格

### 数据准备
在拍摄前，手动设置一些测试数据:
```typescript
// 临时修改 PollenIndexView.ets 的 aboutToAppear()
this.pollenLevel = '中风险'
this.pollenValue = 68
this.currentCity = '广州市'
```

### 避免的问题
- ❌ 截图模糊或有摩尔纹
- ❌ "加载中..."、"获取中..." 等临时状态
- ❌ 网络错误提示
- ❌ 调试信息或开发者选项可见
- ❌ 截图时间戳重复

## 🔄 截图流程

### 步骤 1: 准备设备
1. 在 DevEco Studio 中运行应用到真机
2. 确保应用运行稳定，无崩溃
3. 调整系统时间和电量显示

### 步骤 2: 配置测试数据
1. 打开"区域"页面，设置 1 个置顶城市（如"北京市"）
2. 收藏 2-3 个城市（如"上海市"、"深圳市"）
3. 返回首页，确保数据加载完成

### 步骤 3: 逐页截图
1. 首页 → 音量下键 + 电源键（华为设备）
2. 区域页 → 截图
3. 地图页 → 等待地图完全加载 → 截图
4. 设置页 → 截图
5. （可选）通用设置页 → 截图

### 步骤 4: 导出到电脑
```bash
# 使用 hdc 工具导出截图
hdc file recv /sdcard/DCIM/Screenshots/xxx.png ./screenshots/
```

### 步骤 5: 压缩优化
如果截图超过 1MB，使用工具压缩:
- 在线工具: https://tinypng.com/
- 本地工具: Photoshop / 美图秀秀
- 目标: 保持清晰度，文件 < 1MB

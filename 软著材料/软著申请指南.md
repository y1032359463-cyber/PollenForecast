## 📱 软件功能特点（填写参考）

### 一句话简介
花粉浓度预报是一款为花粉过敏人群提供实时花粉浓度监测、天气预报和健康防护建议的HarmonyOS原生应用。

### 核心功能

1. **花粉浓度监测**
   - 实时显示当前位置花粉浓度（粒/km³）
   - 5级花粉风险等级评估（很低/低/中等/高/很高）
   - 5天花粉浓度趋势预报

2. **天气信息集成**
   - 实时温度、体感温度
   - 风力、气压、湿度
   - 日出日落时间

3. **多城市管理**
   - GPS自动定位当前城市
   - 支持20+热门城市切换
   - 城市收藏、置顶功能

4. **花粉地图**
   - 华为Map Kit地图展示
   - 热门城市花粉标记
   - 智感握姿自适应（单手操作优化）

5. **通知提醒**
   - 花粉预警通知（中度以上）
   - 每日花粉播报（定时推送）
   - 后台代理提醒（应用关闭后仍有效）

6. **桌面卡片**
   - 2×2桌面服务卡片
   - 实时显示花粉浓度
   - 点击快速进入应用

7. **个性化设置**
   - 深色/浅色模式
   - 震动反馈开关
   - 多数据源选择

## 🔧 技术架构

### 开发环境
- **开发工具**: DevEco Studio 6.0.x
- **SDK版本**: HarmonyOS SDK 6.0.0 (API 20)
- **开发语言**: ArkTS
- **UI框架**: ArkUI (声明式UI)
- **应用模型**: Stage模型

### 核心模块
```
PollenForecast/
├── entryability/          # 应用入口
├── pages/                 # 页面
│   ├── Index.ets         # 主页面(Tab容器)
│   ├── GeneralSettingsPage.ets
│   └── NotificationSettingsPage.ets
├── views/                 # 视图组件
│   ├── PollenIndexView.ets  # 花粉指数
│   ├── RegionView.ets       # 区域选择
│   ├── MapView.ets          # 花粉地图
│   └── SettingsView.ets     # 设置
├── service/               # 服务层
│   ├── PollenService.ets    # 花粉API
│   ├── WeatherService.ets   # 天气API
│   └── LocationService.ets  # GPS定位
├── utils/                 # 工具类
│   ├── NotificationService.ets
│   ├── ReminderService.ets
│   └── VibrationUtils.ets
├── model/                 # 数据模型
│   └── PollenModels.ets
└── widget/                # 桌面卡片
    └── PollenCard.ets
```

### 数据来源
- **花粉数据**: Google Pollen API（通过AWS Lambda代理）
- **天气数据**: 和风天气API（通过百度云代理）
